Methodology

A few rules, applied every day. The brief drifts toward noise without them.

Signal vs noise · how inputs are tiered

A claim that moves the trend assessment or threat level requires ≥1 Hard source AND ≥1 independent Tier 1–3 corroboration. Without both, the item goes on the watchlist as "unconfirmed."

Trend assessment · Worse / Same / Better

Window: trailing 72h vs prior 72h.

  • Worse (↑): ≥2 new Hard escalation events with no offsetting Hard de-escalation, OR 1 Hard regime-change event.
  • Same (→): mixed Hard signals, or no Hard signals in trailing 72h.
  • Better (↓): ≥1 Hard de-escalation event AND no offsetting Hard escalation.

Anti-bias defaults · Taleb-grounded
  • Base rates first: deviation from rolling 30-day baseline is the signal, not raw counts.
  • Convexity beats consensus: small-probability × catastrophic-impact tail risks dominate the outlook.
  • Lindy on tensions, anti-Lindy on novelty: long-running patterns continue absent observable physical change. Novelty is regime-change signal.
  • "No news" ≠ "no event": absence is signal only when Tier 1–3 sources are actively reporting nil.
  • Asymmetric updating (Popper): the methodology updates when predictions fail, not when they succeed.
  • No hindsight reframing: once logged, a prediction cannot be retroactively softened.
  • Authority bias check: a head-of-state quote is Soft, not Hard. The brief never quotes a leader as if a quote is an event.

Track record & learning loop

Every brief makes falsifiable predictions. T+1, T+3, T+7 checks score them. Friday calibration runs Brier scores and adjusts source weights and methodology heuristics. Learning is logged, not silent.

How the learning loop works
  1. Each brief logs trend, threat level, watchlist (deadlines + directional implications), and scenario probabilities.
  2. The next morning, the agent scores yesterday's, three-days-ago, seven-days-ago predictions: Hit / Miss / False alarm / Surprise.
  3. On Fridays, a Week-in-Review computes Brier score on probabilities, hit/miss tallies, source reliability.
  4. One concrete methodology delta — heuristic added, removed, or sharpened — logged each Friday in the methodology Changelog. Source weights adjust on the same cadence.
  5. Every change is recorded with date and reason. The agent never silently learns.
